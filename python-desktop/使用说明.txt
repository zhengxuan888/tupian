================================================
    择优臻选出海图片处理器 - 使用说明
================================================

【第一次使用，请按以下步骤操作】

第 1 步：安装 Python
  1) 打开浏览器，访问 https://www.python.org/downloads/
  2) 点击黄色的 "Download Python" 按钮下载
  3) 运行安装程序
  4) 重要！！一定要勾选底部的 "Add Python to PATH"
  5) 点击 "Install Now" 等待安装完成

第 2 步：安装依赖
  双击运行本文件夹里的 "setup.bat"
  等待自动安装完成即可（大约 2-5 分钟）

第 3 步：启动软件
  双击运行本文件夹里的 "run.bat"


------------------------------------------------

【日常使用】

每次使用只需要双击 "run.bat" 即可启动软件。


------------------------------------------------

【功能说明】

1. EXIF 批量写入
   - 上传照片，选择国家和手机型号
   - 自动生成 GPS、拍摄时间等信息
   - 支持批量生成 200 套不同配置
   - 按国家自动分文件夹输出

2. AI 背景合成
   - AI 自动抠图 + 生成新背景
   - 12 种预设背景可选
   - 也可以自己描述想要的背景
   - 注意：需要额外安装 rembg
     命令：pip install rembg

3. 去水印
   - 自动检测水印区域
   - 也可以手动画笔标记
   - 智能修复填充


------------------------------------------------

【常见问题】

问：双击 setup.bat 闪退怎么办？
答：右键 setup.bat，选择"以管理员身份运行"

问：提示"python不是内部或外部命令"？
答：Python没装好，重新安装，记得勾选"Add Python to PATH"

问：AI背景合成用不了？
答：需要额外安装：pip install rembg

问：想打包成EXE发给别人？
答：先安装打包工具：pip install pyinstaller
    然后双击 build.bat，打包好的文件在 dist 文件夹里

问：软件打不开？
答：查看日志文件：
    C:\Users\你的用户名\.ai_image_processor\logs\app.log


------------------------------------------------

【系统要求】

  操作系统：Windows 10 或 Windows 11（64位）
  Python：3.10 或更高版本
  内存：4GB 以上
  硬盘空间：500MB
