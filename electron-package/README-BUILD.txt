============================================================
  AI Image Processor - Windows Build
============================================================

WHAT YOU GET:
  A portable .exe file - no installation needed.
  Just double-click to run. Works offline.

REQUIREMENTS:
  - Windows 10/11 (64-bit)
  - Node.js v18+ (https://nodejs.org/ - LTS version)
  - Internet connection (for one-time setup only)

QUICK START:
  1. Extract this folder anywhere (e.g., D:\AI-Image-Processor)
  2. Double-click "build-windows.bat"
  3. Wait 5-10 minutes (downloads Electron ~80MB)
  4. Find your .exe in the "release" folder
  5. Done! Run the .exe directly.

TROUBLESHOOTING:
  Q: "Node.js is not installed"
  A: Download from https://nodejs.org/ (LTS version)

  Q: "npm install failed"
  A: Run: npm cache clean --force, then retry

  Q: "Build failed" or slow download
  A: The script uses China mirrors automatically.
     If still slow, try a different network.

  Q: .exe blocked by Windows Defender
  A: Click "More info" -> "Run anyway"
     (It's a false positive for unsigned apps)

  Q: Want to rebuild after changes?
  A: Just run build-windows.bat again.

============================================================
  All image processing happens locally in the app.
  Your photos never leave your computer.
